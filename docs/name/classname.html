ClassName命名

ClassName的命名应该尽量精短、明确，必须以字母开头命名，且全部字母为小写，单词之间统一使用破折号 “-” 连接

命名原则

基于姓氏命名法（继承 + 外来），祖先模块不能出现破折号，除了是全站公用模块，如 mod- 系列的命名：

模块命名

全站公共模块：以 mod- 开头

业务公共模块：以 业务名-mod- 开头

常用命名推荐

注意：ad、banner、gg、guanggao 等有机会和广告挂勾的字眼不建议直接用来做ClassName，因为有些浏览器插件（Chrome的广告拦截插件等）会直接过滤这些类名，因此

这种广告的英文或拼音类名不应该出现
